1 — Pendahuluan

1.1 Tujuan Dokumen

Dokumen ini merupakan panduan resmi pengoperasian dan pedoman visual untuk Sistem E-Learning (LMS). Fungsi utamanya:

  1. User Manual — Langkah operasional harian bagi Admin, Trainer, dan Peserta.
  2. UI Guideline — Standar visual (warna, tipografi, komponen) untuk konsistensi pengembangan.
  3. Referensi Tunggal — Seluruh informasi teknis dan operasional dalam satu dokumen.

1.2 Deskripsi Sistem

Platform manajemen pembelajaran berbasis web yang mendukung siklus pelatihan menyeluruh: penyusunan materi, pelaksanaan ujian aman, pemantauan peserta secara langsung, hingga penerbitan sertifikat otomatis.

FiturDeskripsi
Manajemen MateriKonten pelatihan (teks, gambar, video) via editor visual.
Bank Soal & UjianPilihan ganda (otomatis dinilai) dan esai (manual).
Penjadwalan SesiJadwal kelas/ujian dengan status otomatis (Aktif/Selesai).
Safe Exam BrowserPenguncian komputer peserta selama ujian berlangsung.
Live ProctoringSnapshot webcam peserta setiap 30 detik.
Sertifikasi OtomatisPDF sertifikat saat syarat kelulusan terpenuhi.
Analitik & LaporanGrafik tren partisipasi dan rasio kelulusan.

1.3 Target Audiens

  • Administrator — Mengelola seluruh aspek operasional.
  • Trainer — Memantau perkembangan peserta (akses baca).
  • Trainee (Peserta) — Mengakses materi, ujian, dan sertifikat.
  • Tim IT / Developer — Menjaga konsistensi visual dalam pengembangan.

4 — Panduan Penggunaan Sistem

4.1 Akses Sistem (Login)

[ Masukkan Screenshot Halaman Login di sini ]
  1. Buka browser (Chrome / Firefox / Edge versi terbaru).
  2. Masukkan alamat URL sistem yang diberikan administrator.
  3. Isi Username dan Password.
  4. Klik tombol "Masuk ke Dashboard".
  5. Sistem mengarahkan sesuai peran:
    • Admin → Dashboard Administrator.
    • Trainee → Dashboard Peserta.
Lupa Password? Hubungi Administrator untuk reset melalui panel User Management. Fitur reset mandiri belum tersedia.

4.3 Fitur Utama — Panel Administrator

Berikut alur kerja utama Administrator dari awal hingga akhir:

1 Buat Materi Pelatihan

Sidebar → Content → klik "Buat Materi Baru" → isi judul → gunakan editor visual (teks, gambar, video) → Simpan.

[ Masukkan Screenshot Editor Materi di sini ]

2 Buat Bank Soal (Ujian)

Sidebar → Exams → klik "Buat Ujian Baru" → isi parameter:

ParameterKeterangan
Judul UjianNama identitas (contoh: "Ujian Akhir Modul 1").
Durasi (Menit)Batas waktu. Jawaban otomatis terkirim saat habis.
Passing GradeSkor minimum untuk lulus (contoh: 70%).

Setelah simpan, klik "Kelola Soal" → tambah soal pilihan ganda (dinilai otomatis) atau esai (dinilai manual).

3 Susun Modul Pembelajaran

Sidebar → Modules → "Buat Modul Baru" → pasangkan Materi + Ujian → atur urutan → Simpan.

4.4 Fitur Utama — Dashboard Peserta

A. Halaman "Sesi Saya"

[ Masukkan Screenshot Sesi Peserta di sini ]

Seluruh sesi yang diikuti ditampilkan berdasarkan status:

KelompokKeterangan
Sedang BerlangsungKlik untuk akses materi dan ujian.
Akan DatangTerkunci hingga waktu mulai tiba.
Selesai (100%)Semua item telah diselesaikan.
BerakhirSudah melewati batas waktu.

Setiap baris sesi menampilkan Progress Bar (contoh: 3/5 item = 60%).

B. Mengerjakan Sesi

  1. Klik sesi berstatus "Sedang Berlangsung".
  2. Baca Materi — otomatis ditandai selesai setelah dibaca.
  3. Kerjakan Ujian — jika SEB diwajibkan, pastikan aplikasi SEB terinstal.
  4. Selesaikan seluruh item untuk progres 100%.

C. Unduh Sertifikat

Jika progres 100% dan skor ≥ Passing Grade → tombol "Download Sertifikat" muncul → klik untuk unduh PDF.

5 — Troubleshooting & FAQ

Masalah 1: Tidak bisa login — "Login gagal"

PenyebabUsername atau password salah; akun belum terdaftar.
SolusiPeriksa huruf besar/kecil. Pastikan tidak ada spasi. Hubungi Admin untuk reset.

Masalah 2: Monitoring tidak menampilkan foto webcam

PenyebabPeserta belum mulai ujian; webcam tidak diizinkan; SEB tidak aktif.
SolusiPastikan peserta klik "Mulai Ujian". Aktifkan SEB di sesi. Izinkan akses kamera.

Masalah 3: Sertifikat tidak muncul setelah ujian selesai

PenyebabProgres belum 100%; skor di bawah Passing Grade.
SolusiPastikan seluruh materi dibaca. Periksa skor ujian. Hubungi Admin.

Masalah 4: Halaman loading terus-menerus

PenyebabKoneksi lambat; server beban tinggi.
SolusiCek koneksi internet. Refresh (F5). Hapus cache (Ctrl+Shift+Del). Hubungi IT.
